Beschreibung

Beim von Cooke und Wheatstone entwickelten Telegraphen wird beim Sender und Empfänger jeweils der gleiche Buchstabe angezeigt. Jede der 5 Spulen kann nicht bzw. unterschiedlich bestromt werden. Damit können die Zeiger nach rechts, senkrecht oder nach links zeigen. Es können fast alle Buchstaben des Alphabets übertragen werden.
Solange mit der Maus auf einen Buchstaben geklickt wird, wird der entsprechende ASCII Code übertragen. Es müssen von der Steuerung die passenden Wicklungen mit Strom versorgt werden.
Es ist ein Fehler, wenn mehr als zwei Wicklungen bzw. bei einer Wicklung beide Richtungen bestromt werden.

SPS Bezeichnung Kommentar
DigInput.ASCII ASCII ASCII Code
DigOutput.P1R -P1R Zeiger 1 - Rechts
DigOutput.P1L -P1L Zeiger 1 - Links
DigOutput.P2R -P2R Zeiger 2 - Rechts
DigOutput.P2L -P2L Zeiger 2 - Links

DigOutput.P4L -P4L Zeiger 4 - Links
